How to Use the Card App

Steps

  1. Create a new card by signing up with a unique name and your email.
  2. Join an existing card using the card name and your email.
  3. Use "Check" to preview your message and "Submit" to save it.
  4. Click "Flip to Card View" to browse the card, and export it to PDF if needed.

Common Errors

  • Card Already Exists: Pick a different card name when signing up.
  • Case Sensitivity: Card names must match exactly, including upper/lowercase.
